## 技术栈
- **向量数据库**: Qdrant (内存模式)
- **大语言模型**: Ollama (Qwen2.5VL)
- **嵌入模型**: nomic-embed-text
- **编程语言**: Python 3.11
- **依赖管理**: pip + venv
